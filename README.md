# My neovim config 
 
[install neovim](https://neovim.io/doc/install)


## Config path 
### UNIX 
```
~/.config/nvim
````

### Windows 
```
~\AppData\Local\nvim\
```

## Clone Neovim config

```
git clone git@github.com:davidenascimento91/neovim.git

rm ./READEME.md ./.gitignore
```
